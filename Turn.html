Class: Turn

Turn

new Turn(players, environment, societies, relations)

A single turn of the game. Used to maintain history.
Parameters:
Name Type Description
players array an array of functions representing each player's decision mechanism
environment object (optional) the environment before this turn. if undefined, a new environment is created.
societies array (optional) an array of societies. if undefined, or length 0, initializes an array of new societies based on the number of players.
relations array (optional) a 2-dimensional square matrix of relations between societies. if undefined, or length 0, initializes a new matrix based on the number of players.

Methods

process() → {turn}

Takes the following steps in order: 1. get each society's choice given the current environment and relations matrix. 2. modify the environment and all societies according to those choices. 3. modify the environment according to its state. 4. modify each society according to its state. Having updated its environment, societies, and relations matrix, returns the next turn.
